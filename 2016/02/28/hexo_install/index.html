静的サイトジェネレータ[HEXO]を使ってみた

2016-02-28
  • HEXO

HEXOという性的静的サイトジェネレータをインストールして使ってみました。

確認環境は以下の通りです。

  • Mac OS X Yosemite 10.10.5
  • Node.js 5.0.6
  • NPM 3.6.0

  1. インストール

    npmでインストール

    $ npm install -g hexo-cli

    これだけです。-gでグローバルにインストールしています。

  2. プロジェクト作成

    1. $ hexo init [プロジェクト名]

    2. $ cd [プロジェクト名]

    3. $ npm install

  3. ローカルサーバー起動

    1. $ hexo server

    2. ブラウザでhttp://localhost:4000/へアクセス


      が表示されればOK！

  4. 新規記事作成

    1. 雛形の作成

      $ hexo new "[page_name]"

      [プロジェクト名]/source/_posts/[page_name].mdが作成される

    2. 雛形を編集

      1. Front-matterを編集

        -----------
title: 静的サイトジェネレータ[HEXO]を使ってみた ←記事のタイトルを記載
date: 2016-02-28 08:05:10　←生成時に自動で入っていた
tags: HEXO　←タグを記述
-----------

      2. 本文を編集
        先ほどのFront-matter以下が本文となります。
        Markdownが使えるのでとても下記の様に気軽に編集できます。

        ・・・
（Front-matter部分）
-----------
[HEXO](http://hexo.io/)という~~性的~~静的サイトジェネレータをインストールして使ってみました。
1. # インストール
  ## 動作環境
  以下の環境で構築しました。

      3. ブラウザで確認

        http://localhost:4000/
        へアクセスして作成した記事を確認。

以上です。
